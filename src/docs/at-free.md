# 免@设置步骤

> [!DANGER] 重要
>
> - 所有操作必须由**群主**完成，管理员及普通群员无相关权限
> - 按照本文档提供的步骤设置后，官方版提比即可获取群内全部消息，使用方法与旧提比基本相同
> - 开启相关权限后，提比可以获得包括机厅人数查询和被动回复在内的绝大多数旧提比专属功能
> - 如果您的群内同时有官方提比和旧提比，为避免冲突不建议您设置免@

1. 群主在群内点击提比头像，或在群成员列表中点击提比进入提比资料卡

![第一步](/images/step-1.jpg)

2. 点击右上角的齿轮⚙️

![第二步](/images/step-2.jpg)

3. 点击`机器人可获取的群聊消息范围`，并设置为“获取群内消息“

![第三步](/images/step-3.jpg)

4. 勾选 `机器人主动在群聊内发言` (可选)

![第四步](/images/step-4.jpg)

5. 发送`b50`等旧提比指令，查看回复
![第五步](/images/step-5.jpg)
